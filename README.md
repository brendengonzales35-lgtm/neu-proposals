# NEU Proposals

Simple GitHub Pages proposal builder for NEU Electrical Solutions.

## Features
- Editable proposal line items
- Add and delete rows
- Internal cost and selling price
- Automatic totals and estimated profit
- Editable price book
- Browser saving with localStorage
- Print or save as PDF
- Basic sentence parsing for a 20-amp circuit and receptacles

## Publish with GitHub Pages
1. Upload all files and folders to the repository.
2. Open repository Settings.
3. Select Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select the main branch and /root folder.
6. Save.
