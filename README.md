# NEU Proposals Professional Starter

A polished GitHub Pages proposal builder for NEU Electrical Solutions.

## Included
- Professional dashboard layout
- Responsive mobile design
- Editable proposal line items
- Add and delete rows
- Internal cost, selling price, and profit
- Editable price book
- Browser saving with localStorage
- Print / Save as PDF
- Basic smart text entry

## Publish
1. Upload all files and folders to the repository.
2. Open Settings > Pages.
3. Choose Deploy from a branch.
4. Select main and /root.
5. Save.

Replace the NEU text logo with your real logo later if desired.
